# 🛠️ Skill: kb-system-ops (System Operations)

이 스킬은 macOS 시스템의 상태를 모니터링하고, 설정 및 프로세스를 자동화하여 관리하는 기능을 제공한다. 모든 작업은 `jarvis` 독립 계정 내에서 수행되며, 보안 정책을 준수한다.

---

### 1. 주요 기능 (Capabilities)
- **시스템 상태 모니터링**: CPU 사용률, 메모리 점유율, 디스크 여유 공간, 배터리 상태 확인.
- **프로세스 관리**: 특정 프로세스 탐색, 리소스 사용량 상위 프로세스 식별, 안전한 종료 요청.
- **시스템 설정 자동화**: 다크 모드/라이트 모드 전환, 볼륨/밝기 제어, 스크린샷 캡처.
- **네트워크 진단**: 로컬 IP 확인, 활성 포트 스캔, 외부 도메인 핑 테스트.

### 2. 커맨드 명세 (Commands)
- `/kb:sys-stat`: 시스템 요약 정보(CPU, Mem, Disk, Battery) 출력.
- `/kb:sys-top`: 리소스 사용량 상위 5개 프로세스 목록 표시.
- `/kb:sys-capture`: 현재 화면을 스크린샷으로 저장 (`~/Pictures/OpenClaw/` 경로 사용).
- `/kb:sys-theme`: 시스템 테마 전환 (Toggle Dark/Light Mode).
- `/kb:sys-net`: 네트워크 인터페이스 및 활성 연결 상태 확인.

### 3. 보안 및 제약 사항 (Constraints)
- **Sudo 사용 금지**: 시스템 전체 설정을 변경하는 `sudo` 명령은 사전에 정의된 Sudoers 정책에 의해서만 허용된다.
- **격리 원칙**: 타 사용자 계정의 파일 시스템이나 프로세스에 접근을 시도하지 않는다.
- **사용자 승인**: 시스템 재시작, 셧다운, 또는 파괴적인 프로세스 종료 전에는 반드시 사용자에게 확인을 받는다.

### 4. 구현 가이드 (Implementation Notes)
- `system_profiler`, `top`, `df`, `osascript` 등 macOS 내장 CLI 도구를 적극 활용한다.
- 모든 스크린샷 및 로그 데이터는 `jarvis` 홈 디렉토리 내 지정된 폴더에 저장한다.
